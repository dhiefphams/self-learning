# Practice Todo App with Rust and Vuejs

## Requirements
- Node/npm/yarn
- Rust 1.0+

## Build
- `$ git clone https://github.com/dhiefphams/self-learing`
- `$ cd todoapp/rust-vue && npm run build && npm start`

## Development
- `$ git clone https://github.com/dhiefphams/self-learing`
- `$ cd todoapp/rust-vue && npm run dev`
